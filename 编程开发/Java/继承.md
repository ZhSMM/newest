# 继承

## 子类

### 定义

```java
// Father类
public class Father {
    private String name;
    private int age;

    public Father() {
    }

    public Father(String name, int age) {
        this.name = name;
        this.age = age;
    }
    public String say(String string){
        return string;
    }
------setter/getter方法------
    @Override
    public String toString() {
        return "Father{" +
                "name='" + name + '\'' +
                ", age=" + age +
                '}';
    }
}

// 子类Son
public class Son extends Father {
    private String hobby;

    public String getHobby() {
        return hobby;
    }

    public void setHobby(String hobby) {
        this.hobby = hobby;
    }

    public Son() {
    }

    public Son(String hobby) {
        this.hobby = hobby;
    }

    public Son(String name, int age, String hobby) {
        // 继承父类构造器
        super(name, age);
        this.hobby = hobby;
    }

    // 方法覆盖
    @Override
    public String say(String string) {
        return "子类："+super.say(string);
    }

    @Override
    public String toString() {
        return super.toString()+"Son{" +
                "hobby='" + hobby + '\'' +
                '}';
    }

    public static void main(String[] args) {
        Son son=new Son("zhang",12,"baseball");
        System.out.println(son);
        System.out.println(son.say("jhkhk"));
    }
}
```

**父类**：又叫超类、基类，extends表明正在构造的新类派生于一个已存在的类（如Father类），这个已存在的类就叫父类。

**子类**：又叫派生类，如Son类。
